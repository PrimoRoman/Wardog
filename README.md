# Wardog
![app-image](screenshot_of_app.png)
Built a Warhammer game clock using Python. The purpose of this app is to count victory points, command points and time. 

Points for painted armies is added when end game is clicked. If game is resumed points will be deducted until ended again.

## How to use
1. Install the .dmg or .exe file locally
2. Run the app
3. Player 1 goes first (rename accordingly)
4. Counts battle rounds and turns
5. Add points for painted armies when Game is Over
6. Pop up message will declare winner
7. Export CSV to see statistics on time and victory points
